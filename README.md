Machine Learning Model for Cow Selection

Система поддержки принятия решений для оценки потенциальной продуктивности коров перед покупкой.

Задача: снизить риск приобретения низкопродуктивных животных и сделать процесс отбора более объективным.

Решение: разработал модели машинного обучения для прогнозирования удоя и оценки вкусовых качеств молока на основе данных о породе, возрасте, характеристиках корма и происхождении животного.

Результат:

• повысил точность прогнозирования удоя до R² = 0.82;

• снизил среднюю ошибку прогноза до 150 кг молока;

• реализовал алгоритм отбора животных по двум критериям: ожидаемый удой и качество продукции;

• сократил влияние субъективных факторов при принятии решений о покупке.

Стек: Python, Pandas, NumPy, Scikit-learn, Phik, Matplotlib, Seaborn.

Репозиторий: https://github.com/basenok20025/Machine-learning-model-for-cow-selection

Search for Images by Text Description

Система мультимодального поиска изображений по текстовому описанию.

Задача: упростить поиск изображений в больших медиатеках без ручной разметки и сложной системы тегов.

Решение: разработал модель, сопоставляющую текстовые запросы и изображения в едином признаковом пространстве для поиска релевантного визуального контента.

Результат:

• реализовал поиск изображений на естественном языке;

• автоматизировал подбор релевантного визуального контента;

• построил полный ML-пайплайн: подготовка данных, обучение модели и оценка качества;

• сократил время поиска изображений за счёт отказа от ручного тегирования.

Стек: Python, PyTorch, Transformers, Pandas, NumPy, Jupyter Notebook.

Репозиторий: https://github.com/basenok20025/Search-for-images-by-text-description
