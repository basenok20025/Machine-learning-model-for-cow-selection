<h2>Machine Learning Model for Cow Selection</h2>

<p>
Система поддержки принятия решений для оценки потенциальной продуктивности коров перед покупкой.
</p>

<h3>Задача</h3>

<p>
Снизить риск приобретения низкопродуктивных животных и сделать процесс отбора более объективным.
</p>

<h3>Решение</h3>

<p>
Разработал модели машинного обучения для прогнозирования удоя и оценки вкусовых качеств молока на основе данных о породе, возрасте, характеристиках корма и происхождении животного.
</p>

<h3>Результат</h3>

<ul>
  <li>Повысил точность прогнозирования удоя до <strong>R² = 0.82</strong>.</li>
  <li>Снизил среднюю ошибку прогноза до <strong>150 кг молока</strong>.</li>
  <li>Реализовал алгоритм отбора животных по двум критериям: ожидаемый удой и качество продукции.</li>
  <li>Сократил влияние субъективных факторов при принятии решений о покупке.</li>
</ul>
